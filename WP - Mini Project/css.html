The three types of CSS are external, internal, and inline. External CSS is a file that HTML files will link to. Internal CSS is specified at the beginning of an HTML document. Inline CSS is written for a specific element in the HTML document.

CSS applies style rules (defined by selectors and properties) to HTML elements. Furthermore, these rules determine how to set the element's attributes, such as color, font size, margins, etc. CSS Syntax comprises a selector and a declaration block enclosed in curly braces 30
